# SimpleList

OneDrive列表程序，使用WinUI3开发

# 技术选型

WinForms，WPF太老了，Avalonia坑太多了用不明白，HTML+CSS+JS的框架又太大，Iced和Slint生态不知道怎样，不敢上手，遂选用WinUI3，发现写起来很爽，文档也比较完整。第一次写桌面端和WinUI3，权当练手项目了。
